# ur5e-robotic-hand-manual
Operation Manual for the Universal Robots UR5e Collaborative Robot

作成者：CHAN DING KHAI, 堀　煌生

## 日本語 ver

### 概要

<div align="justify">

本マニュアルは、福井工業高等専門学校の卒業研究の一環として作成した、協働ロボット UR5e の操作マニュアルです。

協働ロボットは、安全機能により人と同じ空間で作業できる産業用ロボットの一種で、安全柵なしで導入できることから近年急速に普及が進んでいます。一方で、教育機関において協働ロボットを扱う実践教材はまだ非常に少なく、操作を学べる場自体が限られているのが現状です。

</div>

<sub>出典：https://rsi.jrcnet.co.jp/column-collaborativerobot/</sub>

### 作成の背景

<div align="justify">

前年度の研究にて、協働ロボットの実機操作を通じた教育が学生の学習意欲向上に有効であることが確認された一方、「教材として使用するテキストの構成に改善の余地がある」という課題が明らかになりました。

このマニュアルは、その課題を解決するために新規作成したものです。ロボット操作の予備知識がない学生でも、基本操作を段階的に理解しながら一人で習得できることを目指して設計しています。

</div>

### マニュアルの内容
- UR5e の基本構成・各部の名称
- 電源投入から基本操作までの手順
- 安全確認
- 初心者がつまずきやすいポイントの補足説明

## English ver

### Overview

<div align="justify">

This manual was created as part of my final year research at the Fukui National Institute of Technology, covering step-by-step guidance for operating the UR5e collaborative robot.

Collaborative robots are a category of industrial robots that can safely work alongside humans in the same space, thanks to built-in safety features that remove the need for safety barriers. This has driven rapid adoption of collaborative robots in the production industry. However, hands-on educational materials for collaborative robots aren't common, and opportunities to actually learn how to operate one are still limited.

</div>

### Background

<div align="justify">

Prior research found that hands-on operation of a collaborative robot effectively increased students' motivation to learn, but also identified a problem: the teaching material needed additional content to improve students' understanding.

This manual was created to close that gap. It's designed so that a student with zero prior robotics knowledge can build up basic operating skills step by step, independently.

</div>

### What's Inside
- Basic structure and component names of the UR5e
- Step-by-step procedure from power-on to basic operation
- Safety precautions
- Notes on common points beginners tend to struggle with
